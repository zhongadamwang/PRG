# Task Management

This folder contains individual task files in GitHub issue format for team collaboration.

## Workflow
1. Tasks are created as markdown files following GitHub issue format
2. Files can be imported to GitHub Issues for team collaboration
3. Team feedback is added as issue comments in GitHub
4. Issues can be exported from GitHub to update project documents

## Task File Format
Each task file follows this structure:
- **Filename**: `T##-task-name.md` (aligned with project plan numbering)
- **Content**: GitHub issue format with description, acceptance criteria, and tasks

## Files
- `task-tracking.md` - Overall task progress tracking
- `T##-task-name.md` - Individual task files (created as project progresses)

## Integration
Tasks integrate with:
- GitHub Issues for team collaboration
- Project planning and milestone tracking
- Progress reporting and status updates
